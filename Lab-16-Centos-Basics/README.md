# Lab 16: Package Management RPM

## Rumus Instalasi:
- Download: `wget [URL_PAKET]`
- Install: `sudo rpm -ivh [NAMA_PAKET].rpm`
- Cek Paket: `rpm -q [NAMA_PAKET]`
- Hapus Paket: `sudo rpm -e [NAMA_PAKET]`

## Cara Menjalankan Script:
1. Beri izin: `chmod +x package_manager.sh`
2. Jalankan: `./package_manager.sh install [NAMA_PAKET].rpm`
