# Al Nafi AIOps - Professional Lab Portfolio
**Academic Level:** EduQual Level 6 (AIOps)

This repository contains the technical implementation and automation scripts developed during my EduQual Level 6 studies in AIOps at Al Nafi International College.

## 📌 Project Overview
The focus of this repository is to demonstrate proficiency in Linux administration, shell scripting, and infrastructure automation across different distributions (CentOS and Ubuntu).

## 🛠 Tech Stack
- **Operating Systems:** CentOS, Ubuntu
- **Scripting:** Bash (Shell Scripting)
- **Tools:** RPM, YUM, Git, Nano/Vi

## 🚀 Key Implementations
### 1. Automated Repository Setup
- `setup_repo.sh`: A comprehensive script to automate the creation of local YUM repositories, including metadata generation and `.repo` file configuration.

### 2. Modular Package Management
- `package_manager.sh`: A modular script utilizing Bash functions to streamline the installation and removal of software packages via RPM.

## 📂 Structure
- `/CentOS`: Labs and scripts specific to CentOS environments.
- `/Ubuntu`: Automation tasks for Ubuntu/Debian-based systems.
